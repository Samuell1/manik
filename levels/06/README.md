# Level 06 (Lampas)

**URL:** https://manik.sk/hra/lampas/

## Popis
Na chodbe nesvieti svetlo, lampa je nepouzitelna

## Riesenie
1. 7 pismenova vec ktora chyba v lampasi = SVIECKA
2. SVIECKA bez "I" = SVECKA (6 pismen)
3. Z pismen S-V-E-C-K-A vytvorit dve slova: VEK + CAS
4. Prihlasovacie udaje pre /dielna/: vek:cas

## Subory
- obrazok.jpg - obrazok lampasa
- page.html - HTML stranka levelu

## Prihlasovacie udaje
- Meno: vek
- Heslo: cas
