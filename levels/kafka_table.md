## Možné šifrovacie metódy

### Metóda 1: Písmeno → Kafka pozícia → Atómové číslo
A=1 → Ac → 89
B=2 → Ag → 47
C=3 → Al → 13
D=4 → Am → 95
...

### Metóda 2: Atómové číslo → Kafka pozícia → Písmeno
89 → Ac → pozícia 1 → A
47 → Ag → pozícia 2 → B
13 → Al → pozícia 3 → C
...

### Metóda 3: Číslo = Kafka pozícia → Symbol
1 → Ac
2 → Ag
...
